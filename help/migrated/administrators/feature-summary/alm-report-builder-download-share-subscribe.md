# Report herunterladen, weitergeben und abonnieren

Nachdem Sie einen Bericht im Report Builder gespeichert haben, können Sie ihn nach Bedarf herunterladen, für andere Administratoren in Ihrem Konto freigeben oder den Bericht für einen wiederkehrenden Zeitplan abonnieren.

## Herunterladen eines Berichts

1. Suchen Sie auf der Registerkarte **Berichte** den Bericht, den Sie herunterladen möchten.
2. Wählen Sie **Herunterladen**.
3. Klicken Sie auf **OK**.
   ![](assets/image046.png)
Die Berichterstellung wird asynchron ausgeführt. Sie erhalten eine In-App-Benachrichtigung, wenn die Datei bereit ist.
4. Öffnen Sie die Benachrichtigung und laden Sie die Datei herunter.

>[!NOTE]
>
>Wenn Ihr Bericht keine übereinstimmenden Daten enthält, weil z. B. Filter keine Ergebnisse zurückgeben, wird weiterhin eine leere Datei generiert. Sie erhalten keine Fehlermeldung. Die heruntergeladene Datei enthält Kopfzeilen, aber keine Zeilen.

## Bericht im Report Builder freigeben und abonnieren

Ermöglichen Sie anderen Administratoren den Zugriff auf einen gespeicherten Bericht, und richten Sie die geplante E-Mail-Zustellung in Adobe Learning Manager Report Builder ein.

Das Dialogfeld &quot;**Freigeben und abonnieren**&quot; verfügt über zwei unabhängige Abschnitte: einen, um den Bericht für andere Administratoren freizugeben, und einen, um ein E-Mail-Abonnement einzurichten. Sie können eine oder beide verwenden.

## Bericht für andere Administratoren freigeben

Freigegebene Administratoren können den Bericht anzeigen, duplizieren und bearbeiten.

1. Öffnen Sie den Bericht, den Sie freigeben möchten.
2. Wählen Sie **Aktion** > **Freigeben und abonnieren**.
   ![](assets/image_047.png)
3. Wählen Sie unter **Administratoren mit freigegebenem Zugriff** **Bearbeiten** aus. Wählen Sie dann das Feld **Benutzer/Benutzergruppen auswählen** aus.
4. Suchen Sie nach den Administratoren oder Benutzergruppen, die Sie freigeben möchten, und wählen Sie sie aus.
   ![](assets/image048.png)
5. Wählen Sie **Speichern**.

Die ausgewählten Administratoren können jetzt in der Ansicht &quot;Report Builder&quot; auf den Bericht zugreifen.

>[!NOTE]
>
>Administratoren mit freigegebenem Zugriff können den Bericht anzeigen, duplizieren und bearbeiten. Um den Zugriff zu entfernen, kehren Sie zu **Freigeben und abonnieren** zurück und deaktivieren Sie den Benutzer oder die Benutzergruppe.

## Administratoren für einen Bericht abonnieren

Abonnierte Administratoren erhalten den Bericht per E-Mail in der von Ihnen gewählten Häufigkeit.

1. Öffnen Sie den Bericht, den Sie als Administrator abonnieren möchten.
2. Wählen Sie **Aktionen** > **Freigeben und abonnieren**.
3. Wählen Sie unter **Administratoren mit Abonnement** **Bearbeiten** aus.
4. Wählen Sie die Dropdown-Liste **E-Mail-Häufigkeit**.
5. Wählen Sie eine Häufigkeit:
   * **Täglich senden**
   * **Wöchentlich senden**
   * **Monatlich senden**
6. Suchen Sie im Feld **Benutzer/Benutzergruppen auswählen** nach den zu abonnierenden Administratoren oder Benutzergruppen und wählen Sie sie aus.
7. Wählen Sie **Speichern**.

Abonnierte Administratoren erhalten den Bericht in der ausgewählten Häufigkeit per E-Mail.

>[!TIP]
>
>Wenden Sie mindestens eine Sortierung auf den Bericht an, bevor Sie ein Abonnement einrichten. So wird sichergestellt, dass die Reihenfolge der Zeilen bei jeder geplanten Lieferung konsistent bleibt.
