# Einstellungen

## Übersicht

Adobe Learning Manager bietet detaillierte Administratoreinstellungen, mit denen Sie Ihr Lernmanagementsystem so konfigurieren und anpassen können, dass es den spezifischen Anforderungen Ihres Unternehmens entspricht. Die Einstellungen sind in mehrere Schlüsselkategorien unterteilt, die jeweils unterschiedliche administrative Funktionen erfüllen.

### Grundlegende Einstellungen

* In diesem Abschnitt können Administratoren wichtige organisatorische Details, einschließlich Land/Region, Zeitzone oder Gebietsschema, konfigurieren.
* Konfigurieren Sie Feedback-Mechanismen und Bewertungssysteme für Lernobjekte.
* Verschiedene allgemeine Einstellungen, die sich auf das Verhalten und die Nutzung der Plattform insgesamt auswirken.

Weitere Informationen finden Sie unter [Einstellungen](/help/migrated/administrators/feature-summary/settings/basic-settings.md) in Adobe Learning Manager.

### Integrationseinstellungen

* Konfigurieren Sie Authentifizierungsoptionen einschließlich Adobe ID und Single Sign-on (SSO).
* Verwalten Sie Integrationen mit externen Datensystemen für Benutzerbereitstellung, Kurskataloge und Berichterstellung zur Datensynchronisierung.
* Einrichten und Verwalten von Verbindungen mit anderen Learning Manager-Konten für Inhaltsfreigabe und Initiativen für gemeinsames Lernen

Weitere Informationen finden Sie unter Einstellungen in Adobe Learning Manager .

### Erweiterte Einstellungen

* Erstellen und verwalten Sie benutzerdefinierte Beschriftungen für die Organisation und Kategorisierung von Lerninhalten, sodass die Teilnehmer relevante Materialien leichter finden können.
* Einrichten und Verwalten von physischen Schulungsräumen für Schulungssitzungen mit Kursleiter, einschließlich Kapazität, Ausrüstung und Terminen.
* Konfigurieren Sie Compliance- und Gruppenerfolg-Dashboards.

Weitere Informationen finden Sie unter Einstellungen in Adobe Learning Manager .
