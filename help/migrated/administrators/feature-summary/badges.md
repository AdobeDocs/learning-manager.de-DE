# Abzeichen

Abzeichen sind ein Leistungsnachweis, den sich Ihr Mitarbeiter für den Abschluss eines Kurses verdienen kann. Adobe Learning Manager führt eines der neuesten E-Learning-Konzepte ein, Abzeichen genannt. Fachleute auf der ganzen Welt verwenden diese Abzeichen als Nachweis für das Erlangen bestimmter Qualifikationen oder Lernerfolge.

Sie können Abzeichen definieren, die Benutzern als Motivation dienen.

Administratoren können Abzeichen für Teilnehmer wie folgt erstellen:

1. Melden Sie sich als Adobe Learning Manager als Administrator an.
2. Wählen Sie im linken Fensterbereich **[!UICONTROL Abzeichen]** aus. Es erscheint eine Liste von Abzeichen für Teilnehmer.

   >[!NOTE]
   >
   >Standardmäßig ist eine Liste mit einigen Beispiel-Abzeichen verfügbar.

3. Wählen Sie **[!UICONTROL Hinzufügen]** in der rechten oberen Ecke der Seite aus. Das Dialogfeld &quot;**[!UICONTROL Abzeichen hinzufügen]**&quot; wird angezeigt.

   ![](assets/add-badge1.png)

   *Fügen Sie einen Abzeichennamen und das zugehörige Bild hinzu*

4. Geben Sie den **[!UICONTROL Abzeichennamen]** ein. Laden Sie das Abzeichen hoch, indem Sie auf **[!UICONTROL Abzeichen hochladen]** klicken und auf **[!UICONTROL Speichern]** klicken.

>[!NOTE]
>
>Wenden Sie sich an das Adobe Learning Manager-Supportteam unter [learningmanagersupport@adobe.com](mailto:learningmanagersupport@adobe.com), um nach Abschluss von Kursen oder Lernpfaden benutzerdefinierte Abzeichen oder Zertifikate für Teilnehmer hinzuzufügen.
