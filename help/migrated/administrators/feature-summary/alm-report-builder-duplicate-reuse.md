# Bericht im Report Builder duplizieren und wiederverwenden

Beim Duplizieren eines Berichts werden alle Spalten, Aliase, Gruppen nach Einstellungen, Aggregationen, Filter und das Sortieren in einen neuen bearbeitbaren Bericht kopiert. Verwenden Sie diese Option, wenn Sie eine Version eines Berichts für einen anderen Katalog, eine andere Benutzergruppe oder einen anderen Zeitraum erstellen möchten.

## Bericht duplizieren

1. Öffnen Sie den Bericht, den Sie duplizieren möchten.
2. Wählen Sie **Aktionen** > **Duplizieren**. Eine Kopie des Berichts wird im Bearbeitungsmodus mit dem Namen &quot;[Ursprünglicher Berichtsname]-Kopie&quot; geöffnet. Beispiel: Strukturierte, managerübergreifende Leistungsansicht - Kopieren
3. Geben Sie einen neuen Namen für den Bericht ein.
4. Nehmen Sie Änderungen vor, z. B. aktualisieren Sie Filter, passen Sie Spalten an oder ändern Sie die Sortierung.
5. Wählen Sie **Bericht speichern**.

Der duplizierte Bericht wird auf der Registerkarte **Berichte** gespeichert und ist unabhängig vom Original.

## Vorlage duplizieren

Sie können Vorlagen auch über die Registerkarte **Vorlagen** duplizieren. Durch das Duplizieren einer Vorlage wird ein neuer bearbeitbarer Bericht auf der Registerkarte **Berichte** erstellt. Die ursprüngliche Vorlage bleibt unverändert.

1. Wählen Sie die Registerkarte **Vorlagen** aus.
2. Suchen Sie die Vorlage, die Sie kopieren möchten.
3. Wählen Sie **Duplicate**.
   ![](assets/image049.png)
4. Geben Sie einen Namen ein, nehmen Sie Korrekturen vor, und wählen Sie **Bericht speichern**.

## Best Practices

* Duplizieren Sie einen gut getesteten Bericht als Grundlage, wenn Sie Varianten für verschiedene Teams, Kataloge oder Zeiträume benötigen. Dies ist schneller als das Erstellen von Grund auf neu und gewährleistet eine konsistente Struktur.
* Benennen Sie die Kopie sofort um, um anzugeben, was sie vom Original unterscheidet, z. B. &quot;Compliance by VP - APAC&quot; anstelle von &quot;Copy of compliance report&quot;.
* Aktualisieren Sie nach dem Duplizieren einer Vorlage den Namen, bevor Sie weitere Änderungen vornehmen, damit der Bericht auf der Registerkarte **Berichte** identifiziert werden kann.
